# Use Cases for LED Editor\n\nDerived from quantification: 18 input controls, 16 render feedbacks, 14 features.\n\n1. **Basic editing**: Arrow nav/insert/delete (KEY_LEFT/RIGHT/UP/DOWN/BACKSPACE/DC → visual cursor/line redraw).\n\n2. **Page nav**: PgUp/Dn (scroll adjust → screen refresh).\n\n3. **Line ops**: Home/End/Ctrl+C/X/V (copy/cut/paste line → status update).\n\n4. **Undo/redo**: Ctrl+Z/Y (stack ops → buffer redraw).\n\n5. **Save**: Ctrl+S (file write → modified flag in status).\n\n6. **Search**: Ctrl+? / Enter (regex next → highlight cursor move).\n\n7. **Syntax view**: Toggles F2/F3 (colors/line nums → full re-render).\n\n8. **Status monitor**: Position/time/meter (every update → bottom bar).\n
